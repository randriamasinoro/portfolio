## Attaque padding oracle sur RSA-OAEP

Démonstration d'une attaque padding oracle contre une implémentation vulnérable de RSA-OAEP. L'oracle renvoie un message d'erreur différent selon la validité du padding.

```python
def padding_oracle_attack(ciphertext, oracle, n, e):
    """Récupère le plaintext par requêtes adaptatives à l'oracle."""
    s = 2
    M = {(2 * B, 3 * B - 1)}  # B = 2^(8*(k-2))
    
    while len(M) != 1 or M != {(m, m)}:
        s = find_s(ciphertext, oracle, n, s)
        M = narrow_M(M, s, n, B)
    
    return next(iter(M))[0]
```

<Callout type="info">
  Cette attaque nécessite ~2^20 requêtes oracle en pratique. Les implémentations modernes utilisent des comparaisons en temps constant pour s'en défendre.
</Callout>

- Attaque padding oracle : plaintext récupéré en 18 min sur RSA-2048 vulnérable

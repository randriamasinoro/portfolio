## Contexte

Détection d'anomalies sur un flux de transactions bancaires simulées.

## Architecture

- Ingestion Kafka pour le streaming des transactions
- Isolation Forest pour la détection d'anomalies non-supervisée
- Dashboard Grafana pour le monitoring en temps réel

## Résultats

Taux de détection de 94% sur le jeu de test avec moins de 2% de faux positifs.
